Switch Emulators Guide

January 18, 2023

Ryujinx 指引#

该指南将分为几个部分。您可以使用目录轻松找到您需要的部分。

如果想要更详细的内容，可以直接参考 Ryujinx 的官方文档 here

如果按照本指南进行操作后仍有问题并需要帮助，请加入我们的 Discord 服务器提问.

一、安装模拟器#

(可选) 你可以在 https://ryujinx.org/download 下载到最新的稳定发行版本.

(可选) Github Source (手工编译): https://github.com/Ryujinx/Ryujinx

下载发行版后，只需将其解压缩并运行 Ryujinx 可执行文件即可。或者您可以通过在下载的 Ryujinx 文件夹中创建一个名为「portable」的子文件夹，以便携模式运行 Ryujinx。

二、安装 Keys 和固件（Firmware）#

Keys#

第一次启动 Ryujinx 时，会弹出缺少组件的错误。要解决这个问题，您需要下载 prod.keys，并放到对应的目录。

Keys: 查看

PS： 新的 prod.keys 会包含较旧的密钥。因此只需要只下载最新版即可，并使用 7zip 或 WinRAR 解压缩存档。如果您下载的内容还包含 title.keys，删除即可，不需要用到。

下载 prod.keys 后，打开 Ryujinx，转到左上角并单击「文件」，然后单击「打开 Ryujinx 文件夹」。找到 system 并将（仅）prod.keys 文件放入该文件夹中。

重新启动 Ryujinx，你应该不会再收到错误了。

Firmware#

Firmware: 查看

打开 Ryujinx，转到左上角并单击「工具」，然后在「安装固件」下单击「从 XCI 或 ZIP 安装固件」。导航到固件 zip 文件并单击打开。当系统提示「您是否要安装固件」时，单击「是」。

安装密钥（prod.keys）和固件（firmware）后，您就完成了此步骤。

三、安装游戏、更新以及 DLC#

现在开始安装游戏。下面是一些可信赖的网站列表，可从中获取 Switch 游戏、更新和 DLC。

  • https://nxbrew.com/
  • https://nsw2u.xyz/
  • https://nswgame.com/category/switch/
  • https://www.ziperto.com/nintendo-switch-nsp/

PS： 建议在访问任何这些网站之前安装广告拦截器，例如：https://ublockorigin.com/

PS： 下载时，您可能会看到三种文件类型。 NSP、XCI 和 NSZ。 Yuzu 和 Ryujinx 都不支持 NSZ，所以不要下载。 XCI 已将更新文件合并到游戏文件中，因此您只需下载一个文件。 NSP 有不同的游戏和更新文件，更容易更新。看你自己习惯，喜欢那个就下载哪个。

Ryujinx 加载游戏#

下载游戏后，将其放入您希望保存所有 Switch 游戏的文件夹中（仅放置游戏文件，而不放置更新或 DLC 文件）。

打开 Ryujinx 并点击「选项」，然后点击「设置」。在「常规」选项卡下，找到「游戏目录」并单击「添加」按钮。找到到游戏文件夹所在的位置，然后单击「添加」。最后，点击右下角的「保存」。

加载游戏更新文件 和 DLCs#

打开 Ryujinx。右键单击您要更新或安装 DLC 的游戏。单击「管理标题更新」或「管理 DLC」。然后使用「添加」按钮选择要安装的更新/DLC 文件。如果安装正确，它应该会在游戏旁边显示更新版本。

四、模拟器性能优化#

Ryujinx 不需要很多优化，您只需转到控制选项卡并设置您的控制器（手柄或者键盘，推荐使用 Switch 手柄，减少适配问题）。 Vulkan 是首选，但如果游戏无法在 Vulkan 上运行，请尝试 OpenGL。

PS： 您现在可以开始玩玩游戏，因为所有必需的步骤都已完成。

接下来的部分是可选的，即安装存档（saves）、模组（mods）、着色器（shaders）和一些错误修复。

五、安装模组（Mods）、存档（Saves）和着色器（Shaders）#

Mods#

您可以从以下站点获得一些 Switch Mods：

  • Our Collection
  • TheBoy181’s GitHub
  • Official Yuzu mods page

要安装 Mods，请右键单击要为其安装 Mods 的游戏，单击「打开 Mods 目录」并将 Mods 文件夹粘贴到那里。您必须放置您下载的 mod 文件夹，而不仅仅是其中的文件。

要安装为真实硬件制作的 atmosphere mods，请右键单击游戏，然后单击「打开 Atmosphere Mods 目」并将 Mods 文件夹粘贴到那里。

Shaders#

你可以从我们的 GitHub 找到一些 shader 缓存。

Shaders: https://github.com/JENOVAAbsolute/128BB-Shaders

安装 shaders：

  • 右键单击一个游戏，单击「缓存管理」，然后「打开着色器缓存目录」，并将其替换为您的缓存。

Saves#

你可以在这里找到一些游戏存档：

Our collection: https://drive.google.com/drive/folders/1G2gHYZn7Dbx_X9iYUgpOfCBiWCixBTWU?usp=sharing

Alternative: https://www.homebrewgeneral.net/2020/03/switch-games-save.html

安装存档：

  • 右键单击您要安装存档的游戏，单击「打开应用存档目录」并将存档文件放进去（或者替换）。

备份缓存文件：

如果你需要重新安装 Ryujinx，请转到「%AppData%\Ryujinx\bis\user\save」并备份应用存档文件夹。 如果使用便携模式，请将「%AppData%\Ryujinx」替换为便携文件夹。

六、杂项#

错误或者其他问题修复#

BKTR Error: 当你启动一个更新，而不是实际的游戏时，你会得到这个错误。

Video Core Error: Vulkan驱动程序已经过时了。

  • 如果使用的是 Nvidia GPU, 安装这个文件：https://developer.nvidia.com/vulkan-beta-45836-windows-10-dch
  • 对于 AMD GPU 的，只需在 AMD 软件面板中更新到最新的 *** 可选 *** 驱动程序。

“NCA Header key could not be found” error: 密钥已过期，更新秘钥即可。

Crypto revision error: 密钥已过期（同上）

Updates or DLC failing to install: 密钥已过期（同上）

Games not appearing in your game list

  • 确保游戏文件放入了游戏目录，并且只有游戏文件（更新/存档/Mods 需要放到对应的目录）
  • 您在游戏文件夹中放置了游戏更新而不是游戏。 通过查看文件名来检查这一点。 您将看到[v0]或[v357534]。 [v0]永远都是游戏。 否则就是更新。
  • 游戏文件损坏。
