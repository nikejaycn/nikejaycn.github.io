Synology 215j 安装迅雷远程

发表于 2015-03-09

安装步骤

NAS 准备工作

  1. 开启 Telnet/SSH 服务 (控制面板 -> 终端机和SNMP)

  2. 开启 Web Station 服务 (控制面板 -> WEB服务)

Xunlei Xware 安装

  1. 先去迅雷固件看下你相应的NAS的CPU型号，并下载相应的固件：http://luyou.xunlei.com/thread-12545-1-1.html

    • 215J请下载 Xware_armel_v5te_glibc
    • ATOM CPU请下载 Xware_x86_32_glibc

  2. 打开 Synology 的 File Station，找到 web 文件夹，将刚下载的迅雷固件拉到 web 文件夹下，点击覆盖即可

  3. 右击那个固件，并解压缩（删除原固件文件）

  4. 修改文件夹为xunlei

  5. 打开 Terminal, SSH root@NSA_HOST 到你的 NAS, 密码是你设置的管理密码（Windows 用户用 CMD+Telnet）

  6. 执行下面两句操作后，等待执行完毕

  7. 浏览器输入 http://IP地址:9000/getsysinfo 查看安装状态，具体说明如下：

  8. 进入 http://yuancheng.xunlei.com, 登录你的迅雷帐号来绑定设备激活码

  9. 添加迅雷远程到计划任务 （当机器重启或者升级后，请手动运行一下这个脚本，否则迅雷远程无法启动）
    将任务名称更改为xunlei，并在用户定义的脚本处写入/volume1/web/xunlei/portal
    然后选择计划并将运行平率更改为1小时一次，点击确定。

  10. 迅雷下载会存储在TDDOWNLOAD下，但是这个文件夹在没有第一次迅雷下载之前是隐藏的，必须用迅雷远程下载一个文件后才并重启才可以在共享文件夹中看到。
